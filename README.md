# GGHS Time Table
환영합니다, Rolling Ress의 카루입니다.  
GGHS Time Table(GTT)은 고양국제고등학교 10기 학생들을 위한 학교 생활 도우미 프로그램입니다.

# Features
- GGHS Time Table(GTT): 개인 맞춤 시간표 제작
- GGHS Todo(GTD): 수행평가 일정 관리 프로그램
- GGHS Anonymous: 전교생 대상 익명 채팅방
- Conet: 전교생 대상 커뮤니티

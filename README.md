# GGHS Time Table
고양국제고등학교 10기 전용 학교 생활 도우미 프로그램입니다.

# Features
- GTT(GGHS Time Table): 개인 맞춤 시간표 제작
- GTD(GGHS Todo): 수행평가 일정 관리 프로그램
- Anonymous: 전교생 대상 익명 채팅 프로그램
